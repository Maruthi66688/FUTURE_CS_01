# Vulnerability Assessment Report

## 📌 Internship
Cyber Security Internship – Future Interns

## 🎯 Objective
To perform a vulnerability assessment on a web application and identify security weaknesses based on industry best practices.

## 🔍 Task
Vulnerability Assessment of a Web Application

## 🛠️ Tools Used
- Nmap  
- OWASP ZAP  
- Browser Developer Tools  

## 📊 Methodology
- Planning and Reconnaissance  
- Scanning and Enumeration  
- Vulnerability Analysis  
- Reporting  

## 🚨 Vulnerabilities Identified
- SQL Injection  
- Cross-Site Scripting (XSS)  
- Clickjacking  
- Sensitive Data Exposure  
- Insecure Communication (HTTP)  
- Brute Force Vulnerability  

## ⚠️ Risk Levels
- High  
- Medium  
- Low  

## 🛡️ Recommendations
- Implement input validation  
- Use HTTPS for secure communication  
- Apply proper authentication and authorization  
- Secure sensitive data  
- Enable security headers
